Belajar CSS

Apa itu CSS?

CSS atau Cascading Style Sheet adalah dokumen web yang memiliki fungsi mengatur tampilan dokumen web lain, yaitu HTML. Beberapa contoh pemanfaatan CSS pada HTML adalah untuk mengatur warna latar belakang, gaya huruf, jarak antara elemen, dan masih banyak lagi
CSS terdiri atas kumpulan kode-kode yang keberadaannya tidak menggantikan HTML melainkan pelengkap agar tampilan dokumen HTML menjadi lebih indah dan dinamis. Berikut ini adalah contoh dokumen CSS.

Bagaimana menggunakan CSS?

Ada 3 cara menggunakan CSS:

  1. inline styles: Menuliskan kode CSS langsung pada tag yang akan diubah atau diatur style-nya
  2. Internal/Embedded Styles : Menuliskan kode CSS bagian header dokumen HTML
  3. Eksternal/Linked Styles : Menuliskan kode CSS pada file terpisah dan dipanggil melalui tag link dari dokumen HTML
